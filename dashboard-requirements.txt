# Herramientas de BI Recomendadas para Dashboard de Mantenimiento Vehicular

## Opción Principal: Apache Superset (RECOMENDADO)
- Open source y gratuito
- Integración nativa con PostgreSQL/MySQL
- Mapas interactivos avanzados (integración con Mapbox/OpenStreetMap)
- Dashboards dinámicos con filtros temporales
- API REST para integración personalizada
- Queries cacheables para alto rendimiento
- Alertas automáticas configurables
- Soporte nativo para time series data

## Implementación Sugerida:
1. Mantener el frontend Preact actual para operaciones CRUD
2. Integrar Superset para analytics y reportes
3. Crear vistas materializadas en Django para datos pre-agregados
4. Configurar actualización automática nocturna

## Alternativas Consideradas:
- Grafana: Excelente para monitoring, pero limitado para datos tabulares
- Metabase: Más simple pero menos potente para mapas
- Power BI: Potente pero requiere licencias Microsoft
- Tableau: Muy potente pero costoso
